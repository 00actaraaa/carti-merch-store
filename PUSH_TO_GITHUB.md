# Инструкция по отправке кода на GitHub

Репозиторий уже подключен к: https://github.com/00actaraaa/carti-merch-store.git

## Выполните следующие команды в терминале:

```bash
# Перейти в папку проекта
cd "C:\Users\antip\Documents\carti merch store"

# Переименовать ветку в main (если нужно)
git branch -M main

# Отправить код на GitHub
git push -u origin main
```

Если возникнет ошибка аутентификации, вам нужно будет:
1. Использовать Personal Access Token вместо пароля
2. Или настроить SSH ключи

## Альтернатива: через GitHub Desktop или другой Git клиент

Вы также можете использовать GitHub Desktop или другой графический Git клиент для отправки кода.
